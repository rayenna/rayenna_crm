# Dashboard Module

## Overview

The Dashboard is your central command center, providing a personalized, real-time overview of key business metrics, performance indicators, and visual analytics tailored to your role. ## Filters

### Understanding Dashboard Filters

Filters allow you to customize the dashboard view by selecting specific time periods. This helps you focus on relevant data and analyze performance for specific periods.

### Financial Year Filter

**What is Financial Year**:
- Financial Year (FY) runs from April 1 to March 31
- Format: YYYY-YY (e.g., 2024-25)
- Example: FY 2024-25 = April 1, 2024 to March 31, 2025

**Using Financial Year Filter**:
1. Click **Financial Year** dropdown
2. Available FYs are listed
3. Select one or more FYs by checking boxes
4. Dashboard updates automatically
5. Selected FYs are highlighted

**Filter Behavior**:
- **No Selection**: Shows all data (all FYs)
- **Single Selection**: Shows data for selected FY
- **Multiple Selection**: Shows data for all selected FYs
- **Clear Filter**: Click "Clear Filter" button to reset

**FY Filter Features**:
- Shows all available FYs from your data
- Checkboxes for easy selection
- Visual indication of selected FYs
- Clear filter button when active

### Month Filter

**What is Month Filter**:
- Filters data by specific months within a Financial Year
- Only available when exactly one FY is selected
- Months ordered from April to March (FY order)

**Using Month Filter**:
1. **First**: Select exactly one Financial Year
2. **Then**: Click **Month** dropdown
3. Available months are listed (April to March)
4. Select one or more months by checking boxes
5. Dashboard updates automatically

**Month Filter Behavior**:
- **Disabled**: When no FY selected or multiple FYs selected
- **Enabled**: When exactly one FY is selected
- **No Selection**: Shows all months in selected FY
- **Multiple Selection**: Shows data for selected months
- **Clear Filter**: Click "Clear Filter" button to reset

**Month Filter Features**:
- Automatically disabled when not applicable
- Shows helpful message when disabled
- Checkboxes for easy selection
- Visual indication of selected months
- Clear filter button when active

### Filter Interaction

**How Filters Work Together**:
- **FY Filter**: Primary filter, affects all data
- **Month Filter**: Secondary filter, requires single FY
- **Combined**: Both filters work together when applicable
- **Independent**: Charts may have their own filters

**Filter Application**:
- Filters apply to KPI tiles (metric cards)
- Charts may use unfiltered data with their own filters
- Some charts have independent filter controls
- Filters update data in real-time

**Clearing Filters**:
- **Clear FY Filter**: Clears both FY and Month filters
- **Clear Month Filter**: Clears only Month filter
- **Reset All**: Deselect all filters to see all data
- **Visual Feedback**: Filters show when active

### Filter Best Practices

**Effective Filtering**:
- Start with FY filter for broad view
- Use Month filter for detailed analysis
- Combine filters for specific insights
- Clear filters to see overall picture

**Analysis Workflow**:
1. View all data first (no filters)
2. Apply FY filter for period focus
3. Add Month filter for detailed view
4. Clear filters to compare periods

**Filter Tips**:
- Use filters to focus on specific periods
- Compare different FYs by selecting multiple
- Drill down to months for detailed analysis
- Clear filters regularly to see big picture

## Troubleshooting

### Dashboard Not Loading

**Issue**: Dashboard shows loading or error

**Solutions**:
- Check internet connection
- Refresh browser page
- Clear browser cache
- Try different browser
- Contact administrator if issue persists

### Metrics Not Updating

**Issue**: KPI values not reflecting latest data

**Solutions**:
- Refresh dashboard page
- Check if filters are applied
- Verify data entry is complete
- Clear filters and reapply
- Contact administrator if needed

### Filters Not Working

**Issue**: Filters not applying or not available

**Solutions**:
- Check if FY is selected for Month filter
- Verify filter selections are valid
- Clear filters and try again
- Refresh page
- Contact administrator if issue persists

### Charts Not Displaying

**Issue**: Charts not showing or showing errors

**Solutions**:
- Check internet connection
- Refresh page
- Clear browser cache
- Check if data exists for selected period
- Contact administrator if issue persists
