# User Roles & Permissions

## Overview

The Rayenna CRM system supports multiple user roles, each with specific permissions and access levels.

## Available Roles

### Administrator (ADMIN)
- Full system access
- User management
- All module access
- System configuration

### Sales
- Customer management
- Project creation and management
- Support ticket creation
- Dashboard access

### Operations
- Project management
- Installation tracking
- Support ticket management
- Operations dashboard

### Finance
- Financial data access
- Tally export
- Payment tracking
- Finance dashboard

### Management
- Executive dashboard
- Analytics and reports
- Strategic insights

## Permission Matrix

| Feature | ADMIN | SALES | OPERATIONS | FINANCE | MANAGEMENT |
|---------|-------|-------|------------|---------|-------------|
| Dashboard | ✓ | ✓ | ✓ | ✓ | ✓ |
| Customers | ✓ | ✓ | ✓ | ✓ | ✓ |
| Projects | ✓ | ✓ | ✓ | ✓ | ✓ |
| Support Tickets | ✓ | ✓ | ✓ | - | - |
| Tally Export | ✓ | - | - | ✓ | - |
| Users | ✓ | - | - | - | - |
